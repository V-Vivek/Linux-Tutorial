## chown command
- To change the owner of a file
- Use sudo(Super User Do) prefix if you are a non-root user  
```sudo chown -c new_owner_name filename```

## chgrp command
- To change the group of a file
- Use sudo(Super User Do) prefix if you are a non-root user  
```sudo chgrp -c new_group_name filename```
